## Sobre o Projeto

Este projeto e um jogo da velha direto ao ponto, construído com HTML, CSS e JavaScript puro.

A proposta é simples: abrir no navegador e jogar, sem dependências externas, sem build e sem backend.

## Destaques

- Estrutura mínima de arquivos
- Interface limpa e objetiva
- Interações rápidas com feedback visual
- Lógica de jogo clara e fácil de manter
- Reinício de partida com um clique

## Tecnologias Utilizadas

- HTML5: estrutura da interface e elementos do tabuleiro
- CSS3: layout, efeitos de hover, desenho de X/O e mensagem final
- JavaScript (Vanilla): regras do jogo, alternância de turnos e detecção de vitória/empate

## Funcionalidades

- Iniciar partida automaticamente ao abrir a página
- Alternar turnos entre X e O
- Bloquear células já jogadas
- Detectar vitória por combinações do tabuleiro
- Detectar empate quando não há vencedor
- Exibir mensagem de resultado
- Reiniciar o jogo com o botao de restart
